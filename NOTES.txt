To add local storage:
https://developer.chrome.com/extensions/storage


To "test" the extension in the Developer Kit, use
location.reload(true)

To get contents of msgCount
chrome.storage.sync.get("msgCount", function(item) {console.log(item["msgCount"]);})

Descriptions of files:P
'manifest.json' - extension manifest
'script.js' - code inserted into all pages in FA domain.
'fa.png' - extension icon.
'jquery-1.7.2.min.js' - JQuery source code.
'global.html', 'global.js' - primary interval check code.
'popover.html', 'popover.js' - page displayed on left click
'options.html', 'options.js' - page displayed on right click + "options"
